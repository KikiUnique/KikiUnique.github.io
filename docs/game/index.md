## What is {site.gameName} ##
{site.gameName} is a Live Action Roleplay Game (LARP) where you act the part of a character in a world where the world as we know it was destroyed when the Fey returned to the world.

Before joining the game you create a character picking a faction to be part of, skills you wish to have, and some basic equipment. After a quick talk with a Ref to cover basic knowledge you might need to know you can enter the game area where you act as your character. Undertaking actions and making decisions your character would with the other players and crew as you work together to weave a story.
